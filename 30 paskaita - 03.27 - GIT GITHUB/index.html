Git ir Github

Git - kodo versijavimo programa

GitHub - svetainė, kurioje talpinama versijuptas kodas ir juo dalinamasi tarp programuotojų

Nuoroda i GitHub
  1. Užsiregistruoti į GitHub (susikurti paskyra)
  2. Prisijungti prie GitHub

Komandines eilutes komandos

  • git --version
  • ls - isvardina direktorijas ir failus
  • cd - change directory
  • TAB - uzbaigia rasoma eilute arba pasiulo variantus (nenaudoti tuscio komand line)
  • CTL C - atsaukia vykdoma komanda
  • clear - isvalo terminalo eilute
  • direktorijosKurimas
  • failuKurimas
  • trinimas
  • perkelimas
  • pervadinimas
  • failuKonfiguravimas

Git komandos

  1. init - sukuria git repozitorija
  2. add _nurodytiFailai_ - prideti koreguotus/kurtus/naujus failus/aplankus prie paruostu failu kode versijos kurimui(commitui)
  3. commit -m - is paruostu (staged) failu/aplanku sukuriama kodo versija su atitinkamo zinute (message (-m)), nusakancia kas toje versijoje pakito
  4. status
